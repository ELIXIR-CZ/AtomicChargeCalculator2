AtomicChargeCalculator II

AtomicChargeCalculator II (ACC II) is an application for fast calculation of partial atomic charges. It features 17 empirical methods along with parameters from literature.

Upload structure

Single sdf, mol2, pdb or mmcif file or archive (zip, tar.gz) of those files. Maximum file size: 10 MB

Examples

Cis-homoaconitate

Dissociation

BAX

Apoptotic protein activation

DTP NCI

Set of 500 drug-like molecules from DTP NCI.

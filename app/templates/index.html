AtomicChargeCalculator 2

AtomicChargeCalculator 2 is an application for fast calculation of partial atomic charges. It features 17 empirical methods along with parameters from literature.

Upload structure

Single sdf, mol2, pdb or mmcif file or archive (zip, tar.gz) of those files. Maximum file size: 10 MB

Examples

Tryptophan

One of the amino acids.

3k0h

Small protein.

DTP NCI

Set of 500 drug-like molecules from DTP NCI.
